# Commit-rule

### Commit Type

> 커밋 타입

| Commit Type | Description      |
| ----------- | ---------------- |
| 📑 ::        | 파일 생성        |
| ⚡️ ::        | 기능 업뎃        |
| 🪓 ::        | 기능 삭제        |
| 🐛 ::        | 버그 수정        |
| ⚙️ ::        | 프로젝트 세팅    |
| 🎮 ::        | 테스트 코드 추가 |
| ♻️           | 코드 리팩토링    |

ex)  ⚙️ :: 파이어베이스 프로젝트 세팅
